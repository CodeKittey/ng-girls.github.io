¿Eres una mujer interesada en aprender cómo desarrollar aplicaciones web con las últimas tecnologías? Tenemos buenas noticias ¡Tendremos un taller gratuito para ti! Se llevará a cabo el sábado 24 de Noviembre, en la Universidad UCATEC de la ciudad de Cochabamba, Bolivia.

Creemos que la industria de TI se beneficiará mucho de traer más mujeres a la tecnología. Queremos darte una oportunidad para aprender desarrollo web y convertirte en una programadora.

Durante el taller, construirás y desplegarás tu primera aplicación web con Angular. Angular es una plataforma de código abierto para la creación de aplicaciones web, desarrollada por Google y la comunidad. Ofrece diversas herramientas para crear fácilmente una aplicación dinámica e interactiva que puede ejecutarse en navegadores de escritorio, móviles e incluso robots.

Para participar en el taller necesitarás algunos conocimientos básicos en desarrollo web: HTML, CSS y JavaScript básico o cualquier otro lenguaje de programación.

El taller no tiene costo. Sólo llena el formulario de aplicación.

No esperes demasiado - puedes aplicar hasta el miércoles 14 de Noviembre de 2018
Te invitamos a unirte al grupo de meetup de Angular Bolivia para recibir información sobre futuros eventos y actividades.
